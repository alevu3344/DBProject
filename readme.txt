L'entry point è la classe app/src/main/java/App.java. Affinché il software esegua senza eccezioni, è prima necessario predisporre il database attraverso il file SQL fornito nella root dell'archivio.

E' possibile impostare il nome del database a cui connettersi nella classe App.java.

Viene predisposto un form per registrare un nuovo utente (che può essere un fattorino oppure un consumatore), ma vengono predisposti tre account a fini dimostrativi:

Amministratore: 
  Username: admin         Password: 123

Fattorino:
  Username: fattorino1    Password: 123

Consumatore: 
  Username: consumatore1  Password: 123
